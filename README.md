# MΛ ツ Client

Repositorio oficial de **MΛ ツ Client**, un cliente personalizado basado en DDNet/TClient para DDRaceNetwork.

## Codigo Fuente

El codigo fuente principal esta en:

```txt
M-source/
```

Esta carpeta contiene las modificaciones del cliente MΛ ツ, incluyendo los apartados visuales, recursos, audio, particulas, estelas, HUD, traductor, editor de skins y demas cambios agregados al cliente.

## Descargar Para Jugar

Si solo quieres jugar, descarga el cliente ya compilado desde:

[Releases](https://github.com/thecoderma69/M-client/releases)

En la version 2.0, descarga:

```txt
M-Client-v2.0-win64.zip
```

Luego extrae el `.zip` y abre:

```txt
DDNet.exe
```

## Version Actual

```txt
MΛ ツ Client 2.0
```

## Caracteristicas

- Apartado propio `MΛ ツ`.
- Apartado `Visual`.
- Particulas 3D con varios estilos y reaccion a la musica.
- Estela de tee con estilos, movimiento y reaccion a la musica.
- Efecto musica video con imagen central y modo detras de todo.
- Fondo multimedia para menu y juego.
- Editor de HUD.
- Editor de skins.
- Auto-Reactions.
- Traductor de chat.
- Recursos de audio, cursor y flecha.
- Packs de sonido personalizados.
